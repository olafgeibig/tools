# Tools Monorepo Changelog

This changelog documents the versions of tools contained in each git tag of the `tools` monorepo.

## 0.3.0 — 2025-12-03

Git tag: `0.3.0`

Included tool versions:
- aiproxy: `0.3.0`

Summary:
- aiproxy 0.3.0 introduces a per‑user LaunchAgent install flow (no root), new CLI flags for uninstall and restart, modern `launchctl` usage (`bootstrap`, `enable`, `kickstart`), and log path alignment with `platformdirs`.
